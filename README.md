# VOXIA CONNECTING FUTURES CAMPUS

Welcome to the **VOXIA Portal-PTS**, an interactive **White-Label Presentation Hub** and **Strategic Deck Builder** tailored for Private Universities (PTS).

## 🌟 Overview

This application serves as a dynamic, interactive presentation tool designed to demonstrate strategic concepts such as Early-Engagement, Financial Resilience, and Sustainable Careers for prospective university students. 

### Key Features

*   **Interactive Slide Deck:** A fully navigable presentation mode with engaging UI and animations.
*   **Bento Hub Explorer:** A comprehensive dashboard view summarizing key university programs and metrics.
*   **Live Customizer:** Instantly white-label the presentation by modifying the target institution's name and city.
*   **Simulated Tools:**
    *   **ROI Simulator:** Financial and enrollment recovery calculator.
    *   **AI Portfolio Matcher:** Simulates evaluating student portfolios for academic recommendations.
    *   **Micro-Agency Project Board:** A mockup for assigning real-world industry projects to students.

## 🚀 Tech Stack

*   **Framework:** React 19 + Vite
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS 4
*   **Animations:** Framer Motion
*   **Icons:** Lucide React

## 🛠️ Running Locally

**Prerequisites:** Node.js (v18+ recommended)

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Start the development server:**
    ```bash
    npm run dev
    ```
3.  **View the app:** Open your browser and navigate to `http://localhost:3000` (or the port provided in the terminal).

## 📦 Build for Production

To build the project for production, run:

```bash
npm run build
```
